### Features

- **Custom follow-up interval (Plus)** — Users can choose how often pending confirmation reminders fire (15m, 30m, 1h, 2h, 3h) instead of the fixed 30-minute default
- **How Alarms Work page** — New informational page explaining alarm behavior on iOS

### Bug Fixes / UX Polish

- **Label truncation warning** — Alarm name field warns at 32 characters that text may be truncated on the lock screen
- **"Require Dismissal" rename** — Persistence toggle renamed with footnote explaining iOS lock screen behavior
- **Delete confirmation dialogs** — Improved confirmation UX when deleting alarms
- **Notification tap navigation** — Tapping a notification now correctly navigates to the alarm or category
- **Deleted alarm pre-alarm cancellation** — Deleting an alarm now properly cancels all scheduled pre-alarm notifications
